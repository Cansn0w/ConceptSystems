# Info3315 Concept map feedback system


### Introduction
This project is a Django web server written in Python3, the server is developed for receiving and processing concept maps and produce feedback to learners.

### Setup
To set up the project, install Python3 and all software dependencies from "requirements.txt".
It is adviced to install dependencies withing a Python virtual environment to avoid altering Python from your system (which may be troublesome and hard to manage).
